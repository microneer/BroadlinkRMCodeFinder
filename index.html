Broadlink RM Code Finder

This tool calls the API provided by the RM Bridge Android App so you can use a Broadlink device to read IR codes from many different remote controls and replay them through OpenHAB and the same or different Broadlink devices.

Instructions

  1. Download, install and run the RM Bridge Android App onto a mobile phone or tablet on same network as your Broadlink device.
  2. Press the start button in the RM Bridge app and enter the IP address and port number provided on the app's screen into the boxes below.
  3. Press the Find Broadlink devices button. If all goes well you'll get a list of all your devices in the Broadlink devices section.
  4. Select the device you want to learn with.
  5. Optionally enter a code name in the Code to learn field. This would be the key you want to use for the code in your OpenHAB2 broadlink.map file.
  6. Press the Learn button, and use your remote to send a command at the Broadlink RM device. If all goes well, in a few seconds you'll have the code in the Recorded codes section below.
  7. Enter any other codes and learn them, then copy your newly-learnt codes to your broadlink.map file.
